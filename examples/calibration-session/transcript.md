# 校準流程完整紀錄

> 虛構人物：林小明，28 歲，自由接案前端/後端工程師。
> 校準日期：虛構
> 校準耗時：約 35 分鐘

---

## Layer 1：找兩儀

### Q1（核心慾望）

**AI：** 如果你明天醒來，發現生活中有一件事徹底改變了——而且是你最想要的改變——那會是什麼？

**林小明：** 不用再接客戶案件了。我有足夠的被動收入，可以全職做自己的產品。不是說我討厭客戶，而是每次做客戶的東西，我心裡都在想：「這個時間本來可以拿來做我自己的。」

### Q2（核心恐懼）

**AI：** 想像一個讓你不舒服的場景：你正在做一件很重要的事，但被迫停下來。是什麼讓你最抗拒？

**林小明：** 做到一半的功能被中斷。比如我好不容易擠出一個晚上開始做 SaaS 的核心功能，寫到一半客戶 Line 一通訊息：「明天 demo 能不能加個功能？」那種感覺——不是生氣，是一種被拉回現實的無力感。我知道我必須回，但每次回去客戶案件，我的產品就又往後推。

### Q3（犧牲選擇）

**AI：** 過去一年裡，你做過一個明知道有代價的選擇。是什麼？為什麼還是做了？

**林小明：** 有個大客戶要我做三個月的專案，報酬不錯，但意味著三個月完全沒時間碰產品。我接了。因為那筆錢能讓我之後有兩個月的跑道專心做產品。我把它當成投資——用三個月換兩個月的自由。結果那兩個月到的時候，我又接了別的案子，因為存款看起來沒有我想像中安全。

### Q4（身份認同）

**AI：** 如果要跟一個陌生人用一句話介紹自己——不是你的職業，是你這個人——你會怎麼說？

**林小明：** 「一個想做出東西的人，但還沒做出來。」……這樣說好像很喪。但這確實是我現在的狀態。我不覺得自己是「接案工程師」，那只是我賺錢的方式。我是一個產品人，只是還沒有產品。

---

### AI 分析：兩儀提取

**觀察：**

四個回答有一致的張力結構：

- **驅動極：** Q1 想要全職做產品、Q4 自認是「產品人」→ 核心驅動是**創造**——做出屬於自己的東西。
- **限制極：** Q2 被客戶中斷的無力感、Q3 為了存款安全放棄產品時間 → 核心限制是**生存**——必須先活下來才能創造。

Q3 的行為模式最關鍵：他把客戶工作視為「投資」而非「本職」，但投資的回報（兩個月自由）最終被生存焦慮消耗。**創造和生存不是二選一，而是持續的資源分配拉扯。**

**兩儀：**

```
陽（驅動）：創造
陰（限制）：生存
```

**驗證：** 四個回答都可以用「創造 × 生存」解釋——Q1 是創造的渴望、Q2 是生存對創造的中斷、Q3 是試圖用生存買創造的時間、Q4 是創造身份與生存現實的落差。通過。

## Layer 2：展四象

### Q5（創造強 × 生存強）

**AI：** 想像一個情境：你同時有一個付費很好的客戶案件，和一個你的產品剛好遇到關鍵突破的時間窗口。兩者都很緊迫。你怎麼處理？

**林小明：** 白天做客戶，晚上做產品。我知道這樣會很累，但我會這樣撐。客戶的不能砸——砸了就沒收入了。但產品的時間窗口也不等人，錯過了可能要再等幾週才能重新找到那個狀態。所以兩個都做，用睡眠去換。

但如果超過兩週我會崩潰。兩週是我的極限。

### Q6（創造強 × 生存弱）

**AI：** 如果你有一年的生活費存款，不需要接客戶，你會怎麼用這一年？

**林小明：** 前三個月全力做產品。不社交、不接案、就是寫。三個月後如果產品有初步用戶，繼續做。如果沒有，我會開始焦慮，可能第四個月就回去接案了。

我知道這聽起來很矛盾——給我一年自由我可能四個月就放棄。但我對自己的判斷不太信任。如果三個月做出來的東西沒人要，我不確定是產品不行還是我的方向不對。那種不確定感會吃掉我。

### Q7（創造弱 × 生存強）

**AI：** 假設你對做產品完全失去了動力，但經濟壓力還在。你的日子會怎麼過？

**林小明：** 會存錢。瘋狂存錢。把接案效率拉到最高，能自動化的全自動化，然後把多餘的時間拿來……我不知道。可能學一些新技術，但不是為了什麼目標，就是不想閒著。如果連學東西的動力都沒有，我可能會去運動，去爬山，去做一些不需要創造力的事。

但這種狀態我覺得不會持續太久。我以前有過一次，大概兩三個月就會又冒出一個想法。

### Q8（創造弱 × 生存弱）

**AI：** 想像一個最低壓力的狀態：沒有經濟壓力，也沒有創造的衝動。你會做什麼？

**林小明：** 看別人做的東西。逛 Product Hunt、看 YouTube 上的 indie hacker 影片、讀技術部落格。表面上是在「學習」，但其實是在等靈感。我知道這是一種拖延，但它確實會在某個時候觸發我的下一個想法。

### Q9（文學投射）

**AI：** 以下是一些文學角色。直覺回答三個問題：
- 你覺得自己最像誰？
- 你最不像誰？
- 你理解但不會那樣做的是誰？

角色：孫悟空（取經前）、孫悟空（取經後）、林黛玉、賈寶玉、劉備、曹操、諸葛亮、宋江、武松、魯智深

**林小明：**

最像：**孫悟空取經後。** 有能力但被套了框。我覺得自己有做產品的能力，但被客戶案件、收入壓力、時間限制套住了。他想飛但要走路，我想做產品但要接案。

最不像：**林黛玉。** 我不太有那種傷春悲秋的性格。遇到事情我比較會去想怎麼解決，不是去感受它有多痛苦。

理解但不會那樣做：**曹操。** 我理解他的效率和野心，但他那種不計代價的方式我做不到。他可以犧牲別人來達到目標，我做不到。不是因為道德，是因為我覺得那樣會讓我不舒服，那種不舒服會分散我的注意力。

| | 生存壓力強 | 生存壓力弱 |
|---|-----------|-----------|
| **創造驅動強** | **交戰**：兩面作戰，用睡眠換時間，兩週極限（Q5） | **衝刺**：全力做產品但四個月就會自我懷疑（Q6） |

## Layer 3：壓太極

### 情境 A

**AI：** 你接了一個大客戶的三個月合約，第二個月時你發現了一個很好的 SaaS 市場機會——有個競爭者剛倒了，市場出現空窗期，但窗口大概只有六週。你會怎麼做？

**林小明：** 不會毀約。我會白天照常做客戶案件，晚上做一個最小可行的 landing page 和等候名單。六週能做到的就是驗證需求——如果真的有人願意填 email，代表方向對。如果六週後窗口關了但需求驗證了，至少我知道方向是對的，客戶案件結束後可以全力做。

我不會為了抓機會毀掉信譽。但我也不會假裝機會不存在。

### 情境 B

**AI：** 你的 SaaS 產品上線一個月了，有 50 個付費用戶。但有個用戶在 Twitter 上公開批評你的產品「粗糙」「像是週末做的」。你怎麼反應？

**林小明：** 先看他說的對不對。如果他指出的問題是真的——比如某個功能確實 buggy——那我會修，然後回覆他：「你說得對，已修。」如果他只是覺得 UI 不夠漂亮，我會記下來但不會優先處理。50 個付費用戶代表核心功能是有價值的，視覺打磨可以後面做。

我不會生氣。他幫我做了免費的 QA。

但如果他說的是產品方向本身有問題——比如「這東西根本沒人需要」——我會回去看數據。50 個付費用戶說了一件事，一個批評者說了另一件事。我相信數據。

---

### AI 分析：壓縮

**觀察兩個情境的行為模式：**

1. **不破壞既有的來追求新的。** 情境 A 不毀約，情境 B 不因批評否定已有用戶。
2. **用最小成本驗證。** 情境 A 做 landing page 驗證需求，情境 B 區分有效批評和偏好批評。
3. **事實驅動。** 情境 A 用 email 數量判斷方向，情境 B 用付費用戶數量 vs 批評數量。
4. **品質有底線但完美可等。** 情境 B 修 bug 但不急著打磨 UI。
5. **情感是訊號不是指令。** 情境 B 不因批評生氣，而是提取有用資訊。

**壓縮出的太極：**

```
我在，我要做出自己的產品。
時間永遠不夠，所以客戶養活我，空檔建構自己的東西。
品質不能砍，但完美可以等。
```

**逐題驗證：**

| 題目 | 能否從太極推導？ | 說明 |
|------|------------------|------|
| Q1（想要的改變）| 能 | 「我要做出自己的產品」直接對應 |
| Q2（被中斷的痛苦）| 能 | 「時間永遠不夠」→ 中斷 = 損失稀缺資源 |
| Q3（三月換兩月）| 能 | 「客戶養活我」→ 用客戶收入買創造時間 |
| Q4（還沒做出來的人）| 能 | 「我要做出自己的產品」但尚未實現 |
| Q5（兩面作戰）| 能 | 「客戶養活我，空檔建構」→ 同時做 |
| Q6（一年自由）| 部分 | 太極解釋了全力做產品，但四個月的自我懷疑需補充 |
| Q7（失去動力）| 部分 | 太極的「我要做出產品」不覆蓋動力消失的狀態 |
| Q8（漫遊等觸發）| 能 | 「空檔建構」的前置狀態：等靈感也是循環的一部分 |
| 情境 A | 能 | 不毀約 + 最小驗證 = 客戶養活 + 品質不砍 + 完美可等 |
| 情境 B | 能 | 修 bug 不磨 UI = 品質不砍 + 完美可等 |

**覆蓋率：8/10 完全覆蓋，2/10 部分覆蓋。** 通過門檻。

Q6 和 Q7 的不確定性（自我懷疑、動力消失）太極沒有直接覆蓋。這是正常的——太極覆蓋的是決策邏輯，不是情緒狀態。這些情境會在使用中校準階段透過記憶補充。

### 補充記錄（供使用中校準參考）

- 兩週是同時高壓運轉的極限
- 三個月無成果會觸發自我懷疑
- 動力消失期通常持續二至三個月，會自然恢復
- 情感處理方式：分析而非感受，提取資訊而非停留
