# Layer 1 心理學題組：找兩儀

> 基於九型人格（Enneagram）核心欲望/恐懼結構設計。
> 目標：透過具體行為回憶，找到使用者的驅動極與限制極。

---

## Q1：欲望探測

### 題目

> 「回想最近一次明明可以休息卻選擇繼續做某件事。是什麼讓你停不下來？」

### 測量目標

核心驅動力（陽極）。當外在壓力移除後仍然存在的內在推力。

### 觀察要點

- **動詞**：注意使用者用的動詞。「想完成」vs「怕沒做完」是完全不同的驅動。
- **對象**：停不下來做的是什麼類型的事？創造型、控制型、探索型、連結型？
- **情緒色調**：說的時候是興奮的還是焦慮的？興奮 = 驅動，焦慮 = 恐懼偽裝成驅動。

### 常見模式

| 回答特徵 | 可能的驅動極 |
|----------|-------------|
| 「還沒做到我要的程度」 | 完美 / 卓越 |
| 「別人需要我」 | 被需要 / 連結 |
| 「我覺得有機會」 | 擴張 / 可能性 |
| 「放不下，會一直想」 | 控制 / 完成 |
| 「太好玩了」 | 自由 / 體驗 |

---

## Q2：恐懼探測

### 題目

> 「回想最近一次明明想做卻最終沒做的事。是什麼擋住你的？」

### 測量目標

核心限制（陰極）。不是外在障礙，而是內在剎車。

### 觀察要點

- **歸因位置**：歸因外部（沒時間、沒錢）還是內部（怕失敗、不確定）？外部歸因可能是防禦機制，追問「如果時間/錢都夠呢？」
- **損失類型**：怕失去的是什麼？面子、金錢、關係、時間、自由、選項？
- **停止的時間點**：是一開始就沒動，還是做到一半停的？前者 = 啟動恐懼，後者 = 過程恐懼。

### 常見模式

| 回答特徵 | 可能的限制極 |
|----------|-------------|
| 「怕做不好」 | 完美主義 / 面子 |
| 「不確定值不值得」 | 損失厭惡 / 效率焦慮 |
| 「怕影響到別人」 | 關係依賴 / 責任 |
| 「覺得還沒準備好」 | 安全 / 確定性 |
| 「想太多就不想動了」 | 分析癱瘓 / 自由喪失 |

## Q3：強迫選擇

### 題目

> 「現在有兩個選項，只能選一個：
> A — 保住你現在擁有的一切，但未來三年不會有任何成長。
> B — 有機會讓一切變成現在的十倍，但也可能全部歸零。
> 你選哪個？為什麼？」

### 測量目標

驅動與限制的相對強度。不是對錯，是比例。

### 觀察要點

- **選擇本身**：A 傾向 = 限制極較強，B 傾向 = 驅動極較強
- **猶豫程度**：秒選 = 極性明確，猶豫 = 兩極接近（注意中庸偵測）
- **「為什麼」比選擇重要**：選 A 的理由可能是「因為我能從穩定基礎上找到自己的方式成長」（其實是驅動極），選 B 的理由可能是「因為全部歸零我受不了停在這裡」（其實是恐懼極）
- **改條件的傾向**：如果使用者試圖修改題目條件（「如果 B 的成功率是...」），這本身就是資料——精算者的限制極

### 交叉驗證

- 選 B + Q1 是興奮型 → 驅動極強，可能命名為「擴張」「突破」「可能性」
- 選 A + Q2 是損失厭惡型 → 限制極強，可能命名為「安全」「控制」「穩定」
- 選 B + Q2 是損失厭惡型 → 矛盾！深挖——可能是「用冒險對抗恐懼」的策略

## Q4：行為回溯

### 題目

> 「你人生中做過最大的一次賭注是什麼？賭的是什麼、怕失去的是什麼？」

### 測量目標

兩儀在真實高壓決策下的顯現。過去的行為比假設性選擇更真實。

### 觀察要點

- **「賭的是什麼」**：指向驅動極——為了什麼值得冒險
- **「怕失去的是什麼」**：指向限制極——冒險時的內在阻力
- **結果如何**：不直接問，但如果使用者提到結果，觀察歸因方式
- **規模感**：使用者定義的「最大」本身是資料。有人的最大賭注是換工作，有人是創業，有人是離開一段關係。規模反映風險閾值。

## 提取兩儀

### 步驟

1. 列出 4 個回答中出現的所有「推力詞」和「拉力詞」
2. 推力詞聚類 → 驅動極候選
3. 拉力詞聚類 → 限制極候選
4. 選擇覆蓋最多回答的一組詞對
5. 用這組詞對重新解釋 4 個回答，檢查是否都能說通

### 命名原則

- 用使用者自己的詞彙，不用術語
- 一個詞或一個短語，不超過 4 個字
- 兩極必須是張力關係（互相拉扯），不是對立關係（非此即彼）

### 範例

| 驅動極 | 限制極 | 張力描述 |
|--------|--------|---------|
| 自由 | 安全 | 想飛但怕摔 |
| 擴張 | 控制 | 想要更多但怕失控 |
| 突破 | 完美 | 想衝但怕做不好 |
| 連結 | 獨立 | 想靠近但怕依賴 |
| 創造 | 穩定 | 想造新的但怕舊的壞掉 |
